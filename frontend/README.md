# Eco-Score.AI Frontend

A stunning, modern React application for analyzing e-commerce product sustainability using AI.

## Features

- Premium dark mode UI with crimson/red accents
- Smooth animations with Framer Motion
- Real-time product analysis
- Beautiful bento-box results dashboard
- Fully responsive design

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser

## Build for Production

```bash
npm run build
```

## Backend Integration

The app connects to the backend API at `http://localhost:8000/analyze`

Make sure your FastAPI backend is running before testing the analyzer.
